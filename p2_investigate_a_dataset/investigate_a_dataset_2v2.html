Maybe there are some factors that contribute to the chance of surviving the Titantic. Let's check for a correlation between the variables.

# Calculate the Correlation (Pearson's r)
def correlation(x, y):
    '''
    correlation = average of (x in standard units) times (y in standard units)
    '''
    standard_x = (x-x.mean()) / x.std(ddof=0)
    standard_y = (y-y.mean()) / y.std(ddof=0)
    xy = standard_x * standard_y
    return xy.mean()

Conclusions¶

The Survivors by Class scatter plot shows a negative relationship between the Percentage of Survivors and the Passenger Classes. Addtionally, the CountPlot details the disparity between the Passenger Classes. It it recommended that further statitistical anaylist be performed to measure whether 1st Class passengers have a significatly greater chance of survival than 3rd Class passengers.

Limitations¶

There were several limitations to the exploratory analysis.

One was brought on by missing values. Removing the passengers with missing values could change the correlation between the independent variable and the dependent variable.

Another is a limiation on the conclusions we can draw from the visualizations. Without using statistical testing, we can at best use the visualizations to investigate the variables that might have a correlation visually. Only once a statistic test has been run can any verification on the connection between the variables be implied.

Additonally, there could be other factors not listed in the dataset that could have a strong correlations with survivablity, such as where they were on the ship when it hit the iceberg.
